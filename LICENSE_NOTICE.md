License Notice for HyPER

HyPER (Hybrid Particle-in-cell Electromagnetic Response model) is a hybrid simulation model developed for studying nonlinear wave processes in space plasmas.

The HyPER source code and executable are not publicly distributed. They remain proprietary to the developer unless a separate written agreement is made.

This repository provides documentation, input templates, benchmark descriptions, citation information, and selected post-processing scripts for collaborative academic use.
